VoIP Telecom · Live Play Store

Efone — Real-Time SIP Calling & Chat

Problem & Challenge

Standard HTTP mobile apps fail to handle incoming voice calls when the app is killed or OS battery optimization suppresses background threads.

Engineering Solution & Architecture
  • Engineered an offline-first local data store using ObjectBox DB for instant zero-latency chat access.
  • Built native Android ConnectionService background listeners to intercept incoming SIP INVITE calls in killed states.
  • Developed centralized API interceptor layer managing OAuth 2.0 token auto-refresh and exponential retry policies.
Impact & Result

Achieved 99.8% incoming SIP call delivery rate across background states with zero local message loss.
